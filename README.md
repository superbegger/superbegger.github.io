# 🗺️ 台灣超商/超市 即期地圖 (Discount Map)

這是一個幫助使用者快速尋找附近 **7-11、全家、全聯** 即期折扣商品的 Web 應用程式。透過地圖視覺化的方式，讓使用者能快速尋找想要的即期品。

網站位置 GitHub Pages：[點擊這裡前往](https://superbegger.github.io/)

## ✨ 核心功能 (Features)

* **🗺️ 地圖視覺化**：直覺顯示有即期品庫存的門市位置。
* **🎯 GPS 自動定位**：一鍵取得目前所在鄉鎮，快速尋找附近門市。
* **🔍 條件篩選**：支援關鍵字搜尋與分類選單，尋找特定想吃的商品（例如：飯糰、便當、牛奶）。
* **🏪 門市庫存檢視**：點擊圖釘可查看單一門市的詳細庫存清單與剩餘數量。
* **🔗 快速搜尋同款商品**：在庫存清單中點擊放大鏡 icon，可快速在地圖上尋找同區域內擁有該商品的門市。
* **📱 RWD 響應式設計**：針對手機操作進行深度優化（如 Bottom Sheet 底部抽屜設計），確保極佳的行動端體驗。

## 🛠️ 技術架構 (Tech Stack)

* **前端語言**：HTML5, CSS3, JavaScript (Vanilla JS)
* **地圖套件**：[Leaflet.js](https://leafletjs.com/)
* **地圖圖資**：[OpenStreetMap](https://www.openstreetmap.org/)
* **地理資料轉換**：[TopoJSON Client](https://github.com/topojson/topojson-client)
* **開放資料來源**：台灣鄉鎮市區界線資料取自 [政府資料開放平臺 (data.gov.tw)](https://data.gov.tw/)
* **流量分析**：Google Analytics (gtag)
* **部署平台**：GitHub Pages
* **AI 輔助開發**：程式架構與文案由 Google Gemini 協助優化與生成。
